사용기술: Vue, Vuex, Element UI, Framework7 , Cypress, Nginx

- 그룹웨어의 워크스페이스, 재무, 자산, 모바일 도메인 프론트엔드 개발 및 레거시 코드 개선
- Cypress 기반의 E2E 테스트 도입으로 시스템의 신뢰성과 안정성을 개선
- 무거운 라이브러리를 경량화된 라이브러리로 교체하고, 코드 스플리팅 및 Nginx 캐싱과 gzip 압축을 적용하여 번들 크기를 최적화하고 페이로드를 4.3MB에서 1.1MB로 약 73% 감소시켜 로딩 속도 개선.
- 스켈레톤 UI를 적용하여 다중 앱으로 구성된 그룹웨어의 페이지 전환 시 플리커링 현상을 개선하고 사용자 경험을 향상
- 순서 보장이 필요 없는 API 호출을 병렬 처리로 개선하여 성능 최적화
- 백엔드 개발자와 협업하여 API 응답 속도가 느린 기능을 지속적으로 개선하여 프로젝트의 전반적인 구조 최적화에 기여
- 그룹웨어에 대한 도메인 지식이 없어도 새로운 동료가 쉽게 프로젝트에 적응할 수 있도록 워크스페이스 및 모바일 기능 명세 및 문서화
