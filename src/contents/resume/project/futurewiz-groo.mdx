> Vue, Vuex, Element UI, Framework7 , Cypress

- 그룹웨어의 워크스페이스, 재무, 자산, 모바일 도메인 기능 개발 및 레거시 코드 개선
- Cypress 기반의 E2E 테스트 도입으로 시스템의 신뢰성과 안정성을 개선
- 무거운 라이브러리를 경량화된 라이브러리로 교체하고, 코드 스플리팅을 활용해 번들 크기와 로딩 속도를 개선
- 정적 파일을 캐싱하고, gzip으로 압축된 파일을 클라이언트로 전달하도록 Nginx를 구성하여 모바일 웹 로딩 속도를 개선
- 스켈레톤 UI를 적용하여 다중 앱으로 구성된 그룹웨어의 페이지 전환 시 플리커링 현상을 개선하고 사용자 경험을 향상
- 그룹웨어에 대한 도메인 지식이 없어도 새로운 동료가 쉽게 프로젝트에 적응할 수 있도록 워크스페이스 및 모바일 기능 명세 및 문서화
