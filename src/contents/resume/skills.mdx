- 프론트엔드 개발

  - Next.js, React, Vue 등 다양한 프레임워크와 라이브러리를 활용한 프론트엔드 개발 경험
  - TanStack Query, Zustand, Recoil 등을 활용한 상태 관리 경험
  - React Server Components를 활용하여 클라이언트 번들 크기를 최적화하고 렌더링 성능 개선 경험

- 성능 최적화 및 사용자 경험 개선

  - 코드 스플리팅, 라이브러리 경량화, gzip 압축, 파일 캐싱을 적용하여 페이로드 크기를 최적화
  - 스켈레톤 UI를 적용하여 다중 앱에서 페이지 전환 시 사용자 경험 향상
  - PWA 적용을 통해 스플래시 스크린을 도입하고, 초기 로딩 속도를 개선하여 사용자 경험을 향상

- CI/CD 및 코드 품질 개선

  - GitHub Actions를 활용한 CI/CD 자동화 및 PR 리뷰 알림, 리마인더 기능 구현
  - Cypress 기반 E2E 테스트 도입으로 시스템의 신뢰성과 안정성 개선
  - OpenAPI Generator를 통해 API 스키마 생성을 자동화하여 개발 효율성과 일관성을 향상
